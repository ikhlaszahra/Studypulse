## вљ™пёЏ Step-by-Step Setup in VS Code

### STEP 1: Install Prerequisites

1. **Install Flutter SDK**
   - Go to: https://docs.flutter.dev/get-started/install
   - Choose Windows/Mac/Linux and follow instructions
   - Add Flutter to your PATH

2. **Install VS Code Extensions**
   Open VS Code в†’ Extensions (Ctrl+Shift+X) в†’ Install:
   - `Flutter` (by Dart Code)
   - `Dart` (by Dart Code)
   - `Pubspec Assist` (optional but helpful)

3. **Verify Flutter is working**
   ```bash
   flutter doctor
   ```
   Fix any issues shown (usually Android SDK or Xcode)

---

### STEP 2: Create Flutter Project

```bash
# Open terminal in VS Code (Ctrl + `)
flutter create studypulse
cd studypulse
code .   # Opens the project in VS Code
```

---

### STEP 3: Set Up Firebase

1. **Create Firebase Project**
   - Go to: https://console.firebase.google.com
   - Click "Add Project" в†’ Name it `StudyPulse`
   - Disable Google Analytics (optional) в†’ Create Project

2. **Enable Firebase Services**
   In Firebase Console:
   - **Authentication** в†’ Get Started в†’ Enable `Email/Password`
   - **Firestore Database** в†’ Create Database в†’ Start in Test Mode
   - **Storage** в†’ Get Started (optional)

3. **Install FlutterFire CLI**
   ```bash
   dart pub global activate flutterfire_cli
   ```

4. **Connect Firebase to Flutter**
   ```bash
   # Make sure you're logged in to Firebase
   firebase login
   
   # Inside your project folder:
   flutterfire configure
   ```
   - Select your `StudyPulse` project
   - Select platforms: Android, iOS (or just Android for now)
   - This auto-generates `lib/firebase_options.dart` вњ…

5. **Download config files**
   - Android: Download `google-services.json` в†’ place in `android/app/`
   - iOS: Download `GoogleService-Info.plist` в†’ place in `ios/Runner/`

---

### STEP 4: Get Gemini API Key

1. Go to: https://aistudio.google.com/app/apikey
2. Click "Create API Key"
3. Copy the key вЂ” you'll add it in `lib/services/gemini_service.dart`

---

### STEP 5: Replace Project Files

Copy ALL the `.dart` files from this package into your project's `lib/` folder.
Replace `pubspec.yaml` with the provided one.

---

### STEP 6: Install Dependencies

```bash
flutter pub get
```

---

### STEP 7: Add Your Keys

Open `lib/services/gemini_service.dart` and replace:
```dart
static const String _apiKey = 'YOUR_GEMINI_API_KEY_HERE';
```
With your actual Gemini API key.

---

### STEP 8: Run the App

```bash
# List available devices
flutter devices

# Run on connected Android device or emulator
flutter run

# Run in Chrome (web)
flutter run -d chrome
```

**In VS Code:**
- Press `F5` to run
- Or open Command Palette (`Ctrl+Shift+P`) в†’ `Flutter: Run Flutter`
